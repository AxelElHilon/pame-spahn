🎨 Pamela Spahn - Portfolio Artístico
Sitio web desarrollado para la artista visual Pamela Spahn, donde muestra sus murales, cuadros y obras personalizadas. El objetivo fue crear una experiencia sencilla, visual y fácil de navegar, destacando la personalidad y el estilo artístico de la autora.

🚀 Tecnologías utilizadas
HTML5: Para la estructura semántica de la página.

CSS3: Para los estilos básicos y responsivos.

JavaScript Vanilla: Para funcionalidades simples como animaciones, efecto fade-in y lightbox.

Vite: Usado como herramienta de desarrollo para lograr una carga rápida, hot reload y build optimizado.

⚙️ Instalación y ejecución local

Cloná el repositorio:
git clone https://github.com/AxelElHilon/pame-spahn.git

Ejecutá el proyecto en modo desarrollo:
npm install

Ejecutá el proyecto en modo desarrollo:
npm run dev

Para generar la versión optimizada:
npm run build

📱 Responsive
La web está optimizada para verse correctamente en celulares, tablets y pantallas de escritorio. Se usaron media queries y diseños flexibles para lograr adaptabilidad.
